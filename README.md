Python KVDB
===========

A Simple Key Value database implementation in Python.

- Python 3.8.6


### Setup

$ python -m venv venv
$ pip install -r requirements.txt


### Running

$ python main.py

This will start a REPL. Use 'exit' to exit the REPL.


### Running Tests

$ pytest -v
